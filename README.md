xcoin-spec-cn
=============

中文版的 bitcoin mastercion counterpartyd ethereum bitshares 等的 spec 集合

中文版的spec目前基本网上都有了，不过有些问题，比如不是最新版本，
发现问题没人更新,有些不太准确，这个repo希望可以解决这些问题。
